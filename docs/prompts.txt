Fabric IQ Ontology AI Generator project creation prompt

Create a complete project scaffold for "Fabric IQ Ontology AI Generator".

The application is a frontend UI for business teams to:
- Write a business case in plain English.
- Ask a business Foundry agent to generate a draft ontology without requiring Fabric connection.
- Review business entities and properties.
- Save ontology drafts and submit them to IT.

The application must also support IT teams to:
- Configure Microsoft Fabric Workspace/Lakehouse connections.
- Prompt an IT Foundry agent to generate Lakehouse view mappings and bindings.
- Add/edit/delete bindings and submit bound ontologies.
- Generate ontology package files (`ttl`, `entities.json`, `relationships.json`, `bindings.json`) and save them to blob storage for admin deployment.

Scaffold requirements:
- Angular 17 + Ionic 7 frontend using standalone components.
- Node.js + TypeScript + Express backend.
- Ontology CRUD routes and services.
- Data source configuration routes and services.
- AI generation flow with OpenAI/Azure OpenAI placeholder integration.
- Azure Cosmos DB, Blob Storage, and Fabric/OneLake placeholder services.
- Environment-based configuration.
- Responsive layouts for web, tablet, and mobile.
- Branded header/footer and ontology graph visualization shell.
- Tests for UI services/pages and API health/OpenAI service behavior.
- Documentation covering architecture, setup, environment variables, and MIT licensing.
