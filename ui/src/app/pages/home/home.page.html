Complete Workflow

Phase 1: Author

Business User Role

  1. Open Generate page and describe your domain in natural language
  2. AI generates initial entities, properties, and relationships
  3. Review and refine the ontology graph interactively
  4. Save as draft ontology

Phase 2: Bind

IT User Role

  1. Review business draft in Ontology Editor
  2. Configure Fabric workspace connection in IT Settings
  3. Select available lakehouses and map entities to Fabric tables/views
  4. Validate schema mappings and submit for deployment

Phase 3: Deploy

Admin Role

  1. Review IT-bound ontology and validate all mappings
  2. Approve ontology for production deployment
  3. Deploy TTL + JSON artifacts to Fabric Lakehouse
  4. Monitor deployment status and troubleshoot if needed
